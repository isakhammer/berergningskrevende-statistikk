We first wnat to write an R function that generates samples from a exponential distrobution with rate parameter \(\lambda\). We now that the cumulative exponential distrobution takes the form \[ F(x;\lambda) = 1 - \lambda e ^{-\lambda x}, \quad x \ge 0 \]

Computing the inverse cumulative function exploiting the uniform distrobution \(U \sim Unif(0,1)\) can we get \[ X = F^{-1}(U) = - ln (U)/\lambda \]

We now want to show that this is true using simulation

expdist <- function(rate, n) {
   x <-  -log( (1/rate)*( 1 - runif(n)))/rate
   return(x)
}

n <- 60000
lambda <- 1
plot(expdist(lambda,n))

plot(rexp(runif(n) ,rate=lambda ))
